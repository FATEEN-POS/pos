# Fateen OS 🚀 
### نظام المبيعات والمحاسبة الذكي (المصمم خصيصاً لسوق مصر وأسوان)

**Fateen OS** هو نظام كاشير (POS) وإدارة مخازن متكامل، مبني ليكون خفيفاً، سريعاً، ومتوافقاً مع احتياجات التجار في البيئات التجارية النشطة. يتميز النظام بواجهات عصرية (Dark Mode) تعتمد على لغة HTML البرمجية الصرفة لضمان أقصى سرعة استجابة.

## 🌟 لماذا Fateen OS؟
تم تطوير هذا النظام مع مراعاة التحديات التقنية والتجارية في محافظة **أسوان** وصعيد مصر، حيث يجمع بين سهولة الاستخدام وقوة الربط السحابي.

## ✨ المميزات الرئيسية
* **نظام كاشير فائق السرعة:** واجهة `pos.html` مصممة لعمليات البيع المتتالية مع دعم كامل للباركود.
* **إدارة مخازن ذكية:** مراقبة دقيقة للمخزون، تنبيهات النقص، وإدارة الأصناف عبر `inventory.html`.
* **تحليلات ومحاسبة متطورة:** لوحات تحكم للمديرين والمحاسبين توضح الأرباح، المصروفات، وتقارير المبيعات اليومية والشهرية.
* **تعدد الأدوار (RBAC):** صلاحيات مخصصة (مطور، مدير، كاشير، محاسب، ماركتر) لضمان أمان البيانات.
* **تكامل السحاب (Supabase):** قاعدة بيانات فورية تضمن مزامنة البيانات بين الفروع والمخازن لحظياً.
* **دعم الفاتورة الإلكترونية:** مهيأ تقنياً للربط مع متطلبات مصلحة الضرائب المصرية.

## 🛠 التكنولوجيا المستخدمة
- **Frontend:** HTML5, CSS3 (Custom Properties), JavaScript (Vanilla).
- **Backend:** Supabase (PostgreSQL & Auth).
- **Libraries:** Chart.js (الرسوم البيانية), XLSX & jsPDF (تصدير التقارير), JsBarcode.

## 📁 هيكلية المشروع
* `index.html`: بوابة تسجيل الدخول الموحدة.
* `pos.html`: واجهة نقطة البيع (الكاشير).
* `manager.html`: لوحة التحكم الإدارية.
* `accounting.html`: النظام المحاسبي وإدارة المصروفات.
* `inventory.html`: إدارة المنتجات والمخزون.
* `developer.html`: لوحة التحكم التقنية للمطورين.

## 🚀 كيف تبدأ؟
1. قم برفع الملفات على أي استضافة ويب ثابتة (Static Hosting).
2. قم بإنشاء مشروع على [Supabase](https://supabase.com).
3. قم بتحديث بيانات الربط (URL & Anon Key) في ملفات الـ HTML.
4. ابدأ في إدارة نشاطك التجاري فوراً.

## 🇪🇬 رؤية "أسوان الرقمية"
يهدف المشروع إلى رقمنة التجارة في صعيد مصر، وتوفير حلول تقنية لا تتطلب أجهزة بمواصفات عالية، بل تعمل بكفاءة على أي متصفح ويب، مع دعم فني يفهم طبيعة السوق المحلي.
